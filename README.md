# 🚀 JavaScript Projects

A collection of projects built while learning and exploring modern JavaScript, HTML, CSS, APIs, DOM Manipulation, and frontend development.

This repository serves as a place to practice programming concepts, build projects, and experiment with new technologies.

---

## 📂 Projects

### 🎓 Result Analyzer

A student result analysis application.

**Features**

* Calculate total marks
* Calculate average marks
* Find highest and lowest scores
* Display student performance details

---

### 🌤 Weather App

A weather application powered by the OpenWeather API.

**Features**

* Search weather by city
* Display temperature
* Show humidity
* Weather condition information
* Responsive user interface

---

### 🎬 Movie Info App

A movie search application powered by the OMDB API.

**Features**

* Search movies by title
* View movie posters
* Release year information
* IMDB ratings
* Plot summaries
* Director details
